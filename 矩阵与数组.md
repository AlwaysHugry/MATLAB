## 数组矩阵创建
1.行向量  
```
a=[1 2 3 4]
a=[1,2,3,4]
a =

     1     2     3     4
```
可以用空格与逗号间隔;  
2.多行矩阵  
```
a=[1 2 3 4;5 6 7 8]

a =

     1     2     3     4
     5     6     7     8
```
每一行之间以分号间隔;  

3调用函数创建  
**ones**  
创建全部为1的矩阵
```
a=ones(1,2)

a =

     1     1
```  
**zeros**  
创建全部为零的矩阵
```
a=zeros(1,2)

a =

     0     0
```
**rand**
所有元素在[0,1]的随机数
```
a=rand(3,3)

a =

    0.8147    0.9134    0.2785
    0.9058    0.6324    0.5469
    0.1270    0.0975    0.9575
```

## 矩阵与数组运算
1.MATLAB允许对矩阵所有元素进行简单运算
```
>> a+1

ans =

    1.8147    1.9134    1.2785
    1.9058    1.6324    1.5469
    1.1270    1.0975    1.9575
```
```
>> sin(a)

ans =

    0.7275    0.7916    0.2749
    0.7869    0.5910    0.5200
    0.1266    0.0974    0.8178
```
转置矩阵,使用( ' )
```
>> a'

ans =

    0.8147    0.9058    0.1270
    0.9134    0.6324    0.0975
    0.2785    0.5469    0.9575
```
标准矩阵乘法
```
>> p=a*a

p =

    30    36    42
    66    81    96
   102   126   150
```
逆矩阵inv()
```
>> a=[1 1;1 1]

a =

     1     1
     1     1

>> inv(a)
警告: 矩阵为奇异工作精度。 
```
不可逆矩阵求逆会出现错误;
```
>> a=[-2 1;4 -3]

a =

    -2     1
     4    -3

>> inv(a)

ans =

   -1.5000   -0.5000
   -2.0000   -1.0000
```
matlab存储数字用double型存储,可以通过format命令显示更多位小数;
```
>> format long
>> p=inv(a)

p =

  -1.500000000000000  -0.500000000000000
  -2.000000000000000  -1.000000000000000
```
也可以显示更短
```
>> format short
>> inv(a)

ans =

   -1.5000   -0.5000
   -2.0000   -1.0000
```
**format仅影响显示,并不影响计算;**  
元素级乘法,相对应的元素进行计算 使用 .*
```
>> p=a.*a

p =

     4     1
    16     9
```
同样的除法,求幂都可以加.实现,例如./ .^  
## 串联(连接数组)
串联运算符[]  
水平串联,逗号相隔  
```
>> [a,a]

ans =

    -2     1    -2     1
     4    -3     4    -3
```
垂直串联,分号相隔
```
>> [a;a]

ans =

    -2     1
     4    -3
    -2     1
     4    -3
```
## 数组索引
以a矩阵为例子,
```
>> a=magic(4)

a =

    16     2     3    13
     5    11    10     8
     9     7     6    12
     4    14    15     1
```
1.特定元素
指定行与列下标
```
>> a(4,2)

ans =

    14
```
直接指定一个下标,从头开始下后遍历,注意是一列一列遍历';
```
>> a(12)

ans =

    15
```

2.引用多个数组元素  
指定一个**start:end**的范围即可引用多个数组元素;  
引用前三行的第二列元素--1:3(前三行),2(第二列)  
```
>> a(1:3,2)

ans =

     2
    11
     7
```
单独的冒号指的则是该范围的所有元素;  
引用第三行所有元素--**3,:**
```
>> a(3,:)

ans =

     9     7     6    12
```
我们还可以通过--**start:step:end**--来创建等差数组  
```
>> b=0:20:100

b =

     0    20    40    60    80   100
```
若是省略中间的参数,则默认为等差为1;
